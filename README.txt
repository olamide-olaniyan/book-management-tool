== Description ==

Createa a book management system for your website. This plugin will allow you to manage your books, authors and publishers. You can also add books to your website and display them on your website. 

== Installation ==
1. Upload `book-management-system` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Book Management System to add books, authors and publishers

== Frequently Asked Questions ==
= How do I add a book? =
Go to Book Management System and click on Add New Book
